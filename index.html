THINGS I'M PROUD OF

Top 10 Project in Embedded Systems

May 2019

The final project for my Embedded Systems class was open ended and only required that we wrote in C and that it reflected some of the class material. My partner Jonathan Moon and I implemented an air guitar that generates PWM sound waves to pitches according to buttons pressed and if strum is sensed. I was really tough getting started since we both hardly had any hardware experience. We started the project the earliest, and our first question was asking how to use resistors on the breadboard.

2nd place in Call For Code Hackathon

Sept 2018

Call For Code is a hackathon for natural disaster prepardness. My team and I designed a centralized communication tool for victims and rescuers in the event of a natural disaster that doubles as a weather app for Kenya users.
